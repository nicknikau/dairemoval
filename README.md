# 🚀 PodRip Core Engine

PodRip is a high-performance, automated background utility designed to seamlessly monitor, clean, and validate self-hosted audiobook libraries every 60 seconds.

Built as a native compiled binary and delivered via an isolated Docker architecture, PodRip keeps your media library healthy, optimized, and fully validated without complex system dependencies.

---

## ✨ Key Features

* **Automated background scanning:** Runs silently on a 60-second loop.
* **Zero dependency hell:** `ffmpeg` and dependencies are entirely self-contained inside the container.
* **Microscopic resource footprint:** High-performance, compiled native architecture.
* **Set-and-forget deployment:** Simple single-command installation.
